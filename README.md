# رحلة بناء العوالم الرقمية (Python for Arab Kids)

> **"This project is totally vibecoded."** ✨

مرحبًا بك في **رحلة بناء العوالم الرقمية**، وهي منصة تعليمية تفاعلية مصممة لتمكين الأطفال العرب من دخول عالم البرمجة بلغة بايثون بأسلوب يدمج بين فلسفة CS50 التعليمية وعناصر الألعاب (Gamification).

## 🚀 لماذا هذا المشروع؟
تم بناء هذا المشروع لكسر حاجز الخوف من البرمجة، وتحويلها من مادة جافة إلى مغامرة إبداعية. نحن لا نعلم الكود فحسب، بل نعلم التفكير المنطقي وكيفية صناعة الحلول.

## 🛠️ كيف تم البناء؟ (Tech Stack)
تم بناء المشروع باستخدام تقنيات الويب الحديثة لضمان الوصول السهل دون الحاجة لتثبيت أي برمجيات:

- **المحرك (Engine)**: [Pyodide](https://pyodide.org/) - لتشغيل لغة بايثون حقيقية داخل المتصفح مباشرة.
- **الواجهة (Frontend)**: Vanilla JavaScript مع CSS3 المتقدم لضمان تجربة مستخدم سريعة وسلسة.
- **المحرر (Editor)**: [CodeMirror](https://codemirror.net/) - لتوفير بيئة كتابة كود احترافية.
- **التنسيق (Markdown)**: [Marked](https://marked.js.org/) - لتحويل المحتوى التعليمي من Markdown إلى واجهات تفاعلية.
- **الحالة (State)**: استخدام `LocalStorage` لحفظ تقدم الطالب ومستواه وإنجازاته محليًا.

## ✨ المميزات
- **تعلم تفاعلي**: دروس مبسطة بالعربية الفصحى مع أمثلة حية.
- **نظام مكافآت**: كسب نقاط خبرة (XP)، ترقية المستويات، وفتح إنجازات (Achievements).
- **مهمات برمجية**: تحديات عملية في نهاية كل فصل للتأكد من الفهم.
- **شهادة إتمام**: يحصل "صانع العوالم" على شهادة رقمية عند إكمال الرحلة.
- **بيئة آمنة**: كل شيء يعمل محليًا في متصفحك.

## 🏃 تشغيل المشروع
المشروع لا يحتاج لتثبيت. ببساطة قم بفتح `index.html` في أي متصفح حديث.
أو باستخدام Python لفتح خادم محلي:
```bash
python -m http.server 8000
```
ثم توجه إلى `http://localhost:8000`.
